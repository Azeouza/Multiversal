# Multiversal
The source code for the Multiversal game.
# How to build the game
1.Install Haxe: https://haxe.org/.<br>
<br>
2.Run the following commands in a terminal:<br>
<br>
`haxelib setup`(Press Enter when `Path:` appears)<br>
<br>
`haxelib install lime`<br>
<br>
`haxelib install flixel`<br>
<br>
`haxelib run lime setup flixel`<br>
<br>
`haxelib run lime setup`(Press y when it says `Do you want to install the "lime" command?`)<br>
<br>
`haxelib run flixel-tools setup`(Press y when prompted, enter your PC password if necessary, select the program you use to code and press y again).<br>
<br>
3.Install the program you will use to code:<br>
<br>
Visual Studio Code: https://code.visualstudio.com/<br>
<br>
Sublime Text: https://www.sublimetext.com/<br>
<br>
FlashDevelop: https://github.com/fdorg/flashdevelop<br>
<br>
IntelliJ IDEA: https://www.jetbrains.com/idea/<br>
<br>
(I recommend you use Visual Studio Code).<br>
<br>
4.Install Git: https://git-scm.com/.<br>
<br>
5.Download the game to a folder (e.g., C:\Users\YourName\Documents) in the terminal using the command below:<br>
<br>
`git clone https://github.com/Azeouza/Multiversal.git`.<br>
<br>
6.Finally building:<br>
<br>
Copy the command that you need to run in a terminal to the `Multiversal` folder according to the platform you want to build.<br>
<br>
Windows: `lime build windows`<br>
<br>
Linux: `lime build linux`<br>
<br>
MacOS: `lime build mac`<br>
<br>
Neko (Command that build the Windows version faster): `lime test neko`.<br>
<br>
Tips:<br>
<br>
1.The builds/executables are located in `Multiversal\export\platformname\bin`.<br>
<br>
2.If you want the game to run as soon as it finishes compiling, replace the word `build` with `test` in the command.
