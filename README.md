# Multiversal
O código-fonte do jogo Multiversal.
# Como compilar o jogo
1.Instale o Haxe: https://haxe.org/.
2.Execute os seguintes comandos em um terminal:
`haxelib setup`(Pressione Enter quando aparecer `Path:`)
`haxelib install lime`
`haxelib install flixel`
`haxelib run lime setup flixel`
`haxelib run lime setup`(Pressione y quando aparecer `Do you want to install the "lime" command?`)
`haxelib run flixel-tools setup`(Pressione y quando for perguntado, coloque sua senha do pc se nescessárioro, selecione o programa que você usa para programar e pressione y novamente)
3.Instale o programa que você usará para programar:
Visual Studio Code: https://code.visualstudio.com/
Sublime Text: https://www.sublimetext.com/
FlashDevelop: https://github.com/fdorg/flashdevelop
IntelliJ IDEA: https://www.jetbrains.com/idea/
(Eu recomendo usar o Visual Studio Code).
4.Instale o Git: https://git-scm.com/.
5.Baixe o jogo em uma pasta (Ex.: C:\Usuários\SeuNome\Documentos) no terminal com o comando abaixo:
`git clone https://github.com/Azeouza/Multiversal.git`.
6.Finalmente compilando:
Copie o comando que deve ser executado em um terminal na pasta `Multiversal` de acordo com a plataforma que você deseja compilar.
Windows: `lime build windows`
Linux: `lime build linux`
MacOS: `lime build mac`
Neko (Comando que compila a versão de Windows mais rápido): `lime test neko`.
Dicas:
1.As builds/executáveis ficam em `Multiversal\export\nomedaplataforma\bin`.
2.Se você quiser que o jogo execute assim que ele acabar de compilar, troque no comando a palavra `build` por `test`.
